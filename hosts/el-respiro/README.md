# el-respiro

- Propósito: NAS.
- Hardware: [QNAP TS-253E](https://www.qnap.com/en/product/ts-253e)

  - 1 × 16 TB externo (respaldo general)
- OS: [QTS](https://www.qnap.com/la/operating-system/qts) 5.2
- Redes:
  - `LAN` 192.168.1.3

## Servicios

- Sistema de almacenamiento para la red local.
